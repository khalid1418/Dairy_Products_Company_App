# DairyApp
 Dairy app is a platform to connect between Company to Buyer
 ## Introduction
 Help small Company to publishing there products and help supermarkets and restaurants to see products of company and company price and comment people about company.
 
 
## Features
1-Registration page.

2-Add New Company.

3-Show List Company.

4-Details Company Product.

5-Edit Company.

6-Order Product.

7-Location.

8-Notification.

9-Profile.

10-Setting.





## Technical stuff
1-MVVM Architecture Patterns.
2-Clean Code(DataLayer, DomainLayer , UiLayer).
3-Firebase(FireStore ,Storage, Notification).
4-Google API (Map).
5-Jetpack library(View Model , Live Data , View Binding).
